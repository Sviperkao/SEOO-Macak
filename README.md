# SEO Mačak — Lokalni scaffold

Ovo je jednostavan statički scaffold sajta sa sledećim stranicama:

- `index.html` — Početna
- `seo-optimizacija.html` — SEO Landing
- `izrada-sajtova.html` — Izrada sajtova i Dizajn
- `blog.html` — Blog / Case Studies
- `kontakt.html` — Kontakt (formular)

Kako pokrenuti lokalni preview:

```bash
# iz root foldera workspace-a
python3 -m http.server 8000
# zatim otvori http://localhost:8000
```

Ako želite, mogu:
- povezati domen i pomoći s deploy-om (Netlify / Vercel / GitHub Pages)
- doraditi dizajn i sadržaj
